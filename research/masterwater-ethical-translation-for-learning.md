## Ethical translation for AAE/C2Acct

Allowed use in this repo:
- sequence the learning path clearly
- reduce learner overwhelm through staging
- use visible progress to support completion
- use repetition for retention
- keep tone calm, competent, and serious
- create confidence through clarity and predictable structure

Not allowed in this repo:
- coercion
- intimidation
- false urgency
- manipulative pressure
- regulated-learning theater
- tactics designed to override learner judgment

## Recommended AAE implementation pattern

- open each module with a clear reason and expected output
- keep reading, quiz, and assessment steps in a fixed order
- make the next step obvious after each checkpoint
- explain unlocks as earned evidence, not surprise rewards
- use neutral, professional reminders rather than countdown pressure

## Boundary statement

MASTERWATER should influence learning flow and completion support only. It should not be used as a persuasion playbook for accounting learners, regulated-profession users, or authorization-sensitive platform actions.
