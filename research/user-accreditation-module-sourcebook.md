# User Accreditation Module Sourcebook

Accessed: 2026-03-16

## Official-source facts

- NASBA fields of study provide the cleanest canonical bucket system for accounting-adjacent professional learning. Source: https://www.nasbaregistry.org/resources/fields-of-study
- NASBA/AICPA/QAS CPE Standards emphasize documented instructional methods, participant evidence, and retention. Source: https://www.nasbaregistry.org/sponsor/site/docs/CPEStandards.pdf
- State boards such as New York, Texas, Florida, Illinois, Virginia, and California all reinforce that competence maintenance is jurisdiction-sensitive and evidence-sensitive. Sources:
  - https://www.op.nysed.gov/professions/certified-public-accountants/mce-questions-answers
  - https://www.tsbpa.texas.gov/php/fpl/options/CPEGuidelines.html
  - https://www2.myfloridalicense.com/cpa/documents/CPEGuidelines.pdf
  - https://idfpr.illinois.gov/content/dam/soi/en/web/idfpr/faq/dpr/cpa-cpe-faqs-6-28-18.pdf
  - https://boa.virginia.gov/wp-content/uploads/2025/07/CPE-Informational-Handout.pdf
  - https://www.dca.ca.gov/cba/licensees/ce-requirements.shtml

## University and society guidance

- AACSB standards support measurable learning goals and continuous-improvement loops. Source: https://www.aacsb.edu/accredited/standards/business
- Rutgers and Kelley both show that rigorous accounting learning can be modular, stackable, technical, and career-linked. Sources:
  - https://www.business.rutgers.edu/stackable
  - https://myrbs.business.rutgers.edu/undergraduate-newark/accounting-minor
  - https://kelley.iu.edu/about/directory/offices/instructional-consulting/assessment/accounting.html
  - https://kelley.iu.edu/programs/ms-accounting/index.html
- IIA, IMA, and ISACA provide credible examples of annual or rolling continuing-education maintenance, ethics requirements, and audit/attestation models. Sources:
  - https://www.theiia.org/en/certifications/cia/cpe-requirements/
  - https://www.theiia.org/en/standards/2024-standards/future-of-the-ippf-evolution/ethics-and-professionalism/
  - https://www.imanet.org/en/IMA-Certifications/CMA-Certification/Maintain
  - https://www.isaca.org/credentialing/cgeit/maintain-cgeit-certification

## Supported posture for AAE users

- internal professional learning
- accreditation-aligned educational framework
- CPE-ready architecture
- state-configurable compliance path

## Unsupported posture for AAE users

- not an approved CPE credit issuer
- automatic CPE acceptance in any state
- equivalence to university credit or credential maintenance without sponsor validation

## Recommended user-module pattern

1. Reading
2. Short quiz
3. Module assessment
4. Cumulative final assessment
5. Evidence-backed unlock
6. Exportable completion record

## Evidence expectations for serious use

- source citation per question
- answer rationale
- assessment version ID
- completion timestamp
- learner identity and role context
- score and pass/fail outcome
